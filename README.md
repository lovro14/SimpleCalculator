## Simple calculator options
* Only one mathematical operation per request is allowed
* Mathematical operators: +,-,*,/
* Calculator can resolve mathematical expressions like:
  ** 3+4
  ** 3*6
  ** -4*-3
  ** 3/-4
  ** 3-9
  ** -3+4
